myWMS LOS

myWMS LOS is an open source warehouse management system. It is developped by LinogistiX GmbH, Dortmund (Germany) and published under GNU GPL licence.

myWMS LOS offers ready to use business processes for warehouse management and a graphical user interfaces based on the Netbeans Rich Client Platform.

This help set includes context sensitive help for the dialogs.

For more information, add-on modules or project requests please visit our web site (http://www.linogistix.com) or the myWMS LOS Wiki (http://wiki.linogistix.com).
