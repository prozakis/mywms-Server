Importieren von Artikeln

Über den Menüpunkt "Datei-Importieren-Artikel" können XML Dateien im Microsoft Excel 2003 XML Format eingelesen werden. Eine Vorlage findet sich hier.

Vorgehen:

  1. Herunterladen der Vorlage
  2. Öffnen der Vorlage in Microstoft Excel, OpenOffice.org Calc oder in einem XML Editor
  3. Bearbeiten der Datei (Pro Artikel einen Zeile) und speichern (im Microsoft Excel 2003 XML Format!)
  4. Importieren dieser Datei

Der Mandant, dem der Artikel gehört

Der Name des Artikels
