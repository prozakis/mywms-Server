LOS Datenimport

LOS Datenimport ist ein Modul zum Importieren von Stammdaten. Die einzelnen Importfunktionen sind über das Menü "Datei-Import" zu erreichen.

Importiert werden XML Dateien im Microsoft Excel 2003 XML Format. Die Dateien können mithilfe eines Tabellenkalulationsprogrammes (Microsoft Excel oder OpenOffice.org Calc) erzeugt bzw. bearbeitet werden.
