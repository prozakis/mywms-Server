Transfer a Unit Load

The Wizard allows for transferring a unit load to a given storage location.

Step 1: Choose destination

  • Unit Load: Choose a Unit Load that should be transferred
  • Destination location: Choose a Storage Location where the Unit Load should be placed
  • Ignore Lock of location option: Normally, if a destination location is locked no Unit Load can be transferred to this location. If this option is set, the lock of the location is ignored, i.e. the Unit Load can be transferred to this location nonetheless.
  • Remove reservation: Not supported at the moment.

Step 2: Provide a hint

Here, the user can enter some additional information indicating why a manual transfer of the unit load has been necessary. The entered text appears in the Unit Load records.
