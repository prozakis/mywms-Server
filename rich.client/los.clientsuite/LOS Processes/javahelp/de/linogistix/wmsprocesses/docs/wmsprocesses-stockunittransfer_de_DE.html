LHM umbuchen

Dieser Dialog dient in der Regel einem "Aufräumen" im Lager.
Zur Umbuchung von Warenmengen auf ein anderes Ladehilfsmittel unter Menüpunkt "Aktionen" die Unterpunkte "LHM umbuchen", "Lagereinheit umbuchen" anwählen.

Lagereinheit umbuchen

Der Menüpunkt "Lagereinheit umbuchen" öffnet einen Dialog, mit dem einer Lagereinheit ein anderes Ladehilfsmittel zugeordnet wird. Dies kann zum einen durch Umpackung oder Verräumung im Lager notwendig sein.

Detaillierte Erläuterung der im Dialog verwendeten Begriffe:

Lagereinheit
Geben Sie die umzubuchende Lagereinheit ein.
Reservierung der Lagereinheit löschen
Die Reservierung der Lagereinheit, beispielsweise durch eine aufgegebene Warenbestellung, wird aufgehoben.
Lagereinheit entsperren
Die Sperrung der Lagereinheiten, beispielsweise durch eine Chargengültigkeit, wird aufgehoben.
Ziel LHM
Das Ladehilsmittel, auf welche die Lagereinheit umgepackt und gebucht wird.
