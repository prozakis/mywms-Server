Avis erzeugen

Mit diesem Dialog wird ein Avis erzeugt.

Der Prozess Avis bedeutet, dass das LVS einen bestimmten Artikel von einem ausgezeichneten Mandanten zu einem angegebenen Zeitpunkt erwartet. Zusätzlich besteht die Möglichkeit, diesen Artikel einer bereits bestehenden Charge zuzuordnen, oder eine neue zu erstellen. Abschließend wird der Auftrag bei Klick auf die Schaltfläche "Auftrag erzeugen" im System hinterlegt, also avisiert.
