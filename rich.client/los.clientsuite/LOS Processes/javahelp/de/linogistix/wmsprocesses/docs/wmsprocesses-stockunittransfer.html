Transfer Stock Unit

With this dialoge you can manually transfer a Stock Unit from one Unit Load to another. This action can be invoked via "Actions-Transfer Stock Unit"
Stock Unit
The Stock Unit that should be transferred
Remove reservation
Remove an existing reservation of the Stock Unit, that has been reserved e.g. because of an order
Unlock Stock Unit
Remove an existing lock of the Stock Unit, that has been locked.
Destination Unit Load
The Unit Load to which the Stock Unit should be transferred.
