Release notes - myWMS LOS - Release 1.3

Major changes since release 1.2

Bug fixes

  • There are several bug fixes. Have a look to the subversion repository for details.

Functional enhancements

General

  • Java Version 1.6 is now supported

  • The start of the Client with Java Webstart is possible



Mobile

  • New: Info Dialog

  • New: Receiving with direct storage

  • New: Receiving without advice and direct storage

  • Enable direct storage on fixed locations (usable for manual replenishment)



Masterdata

  • New: Bills of material

  • Enharncement of SystemProperties: Properties can be defined for clients and workstations

  • Added some measurement attributes to different kinds of master data.

  • Goods receipt. Print labels on workstation printer (rich client only)



GUI

  • Show more infos in some dialogs (Transfer Unit Load / Stock Unit and Change Amount)

  • Show number and name of item data in the dialogs

  • Changes to some default search values and entity actions



User Exits

  • Added UserExits to write stock changes to external software

  • Added UserExits to write finished picking orders to external software

  • Added UserExits to write finished goods receipts to external software



Webservices

  • New: web-service to write Advices into LOS

  • New: web-service to write BOMs into LOS

  • Added a list of EAN-Codes to the web-service of item data
